# Requesting camera access ...

iOS (Safari):
  1. Open Settings → Safari
  2. Tap Camera → Set to "Ask" or "Allow"
  3. Return to Safari and reload the page
Android (Chrome):
  1. Tap the 🔒 lock icon in the address bar
  2. Select Permissions or Site settings
  3. Set Camera to "Allow"
  4. Reload the page
Please check your browser settings and allow camera access.
